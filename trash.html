Flexbox Layout

display: flex

Demo 1 — Navbar Horizontal

navbar-flexbox.html
Brand.
  • Home
  • About

Konsep: Navbar menggunakan display: flex dengan justify-content: space-between untuk mendistribusikan tiga elemen (logo, menu, tombol) secara merata di sepanjang sumbu horizontal. align-items: center memastikan semua elemen sejajar secara vertikal di tengah. Menu navigasi di dalamnya juga menggunakan Flexbox dengan gap untuk jarak antar item.

Konsep: "Holy Grail Layout" adalah pola klasik tiga kolom dengan header dan footer. Diimplementasikan menggunakan Flexbox bersarang: kontainer luar menggunakan flex-direction: column untuk menyusun header, konten tengah, dan footer secara vertikal. Di dalam konten tengah, flex-direction: row (default) menyusun dua sidebar dan konten utama secara horizontal. Konten utama diberi flex: 1 agar mengisi sisa ruang yang tersedia.

CSS Grid Layout

display: grid

Demo 4 — Grid 3 Kolom Dasar

basic-grid.html
grid-item 1
grid-item 2
grid-item 3
grid-item 4
grid-item 5
grid-item 6
Konsep: Grid tiga kolom dibuat dengan grid-template-columns: repeat(3, 1fr) — sintaks repeat(3, 1fr) membuat tiga kolom dengan lebar yang sama, masing-masing mengambil satu bagian dari ruang yang tersedia (fractional unit). Item akan mengisi sel secara otomatis dari kiri ke kanan, atas ke bawah, tanpa perlu menentukan posisi secara manual.

magazine-grid.html

Konsep: Layout majalah dibuat dengan grid dua kolom (2fr 1fr). Artikel utama diberi grid-row: 1 / 3 agar membentang sepanjang dua baris, menciptakan hirarki visual yang jelas. Dua artikel sampingan mengisi sel-sel di kolom kanan secara normal. Teknik ini menunjukkan kekuatan Grid dalam mengatur tata letak dua dimensi (baris dan kolom secara bersamaan).

03

Responsive Design

@media queries

Demo 7 — Grid Responsif Tanpa Media Query (auto-fill)

responsive-grid.html
Item A
Item B
Item C
Item D
Item E
Item F
Konsep: Menggunakan repeat(auto-fill, minmax(120px, 1fr)) — browser akan secara otomatis menentukan jumlah kolom berdasarkan ruang yang tersedia, dengan lebar minimum 120px per kolom. Di layar lebar muat 6 kolom; di layar kecil otomatis menjadi 2 atau 1 kolom. Ini adalah teknik responsif murni tanpa satu pun @media query.

Konsep: Layout yang secara eksplisit mendefinisikan tiga breakpoint melalui @media query: empat kolom untuk desktop (≥840px), dua kolom untuk tablet (≤840px), dan satu kolom untuk smartphone (≤480px). Coba ubah ukuran jendela browser untuk melihat perubahan layout secara langsung. Pendekatan ini memberikan kontrol penuh atas tampilan di setiap ukuran layar.
